# @bingumd/sass-utils

Sass plugin to add a collection of mixins, helpers and tools for your saas.

## :hammer: Installation

##### Yarn
```sh
yarn add @bingumd/sass-utils
```

##### Npm
```sh
yarn install @bingumd/sass-utils
```

## :gear: Usage

```scss
@use '@bingumd/sass-utils' as util;
```
